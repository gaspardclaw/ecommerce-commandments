# Les 100 Commandements du E-commerce

Une collection satirique et cynique des vérités non-dites du e-commerce français.

## Edition Preview - 20/100

Cette édition preview contient les 20 premiers commandements d'une collection qui en contiendra 100.

### Style
- Illustrations indie/zine avec esthétique punk
- Encre noire sur papier crème
- Bords rugueux, texture photocopiée
- Humour cynique d'initié

### Commandements inclus

Les vérités que tout professionnel du e-commerce a vécues :
- Les budgets tests vs bugs
- La documentation perdue avec Jean-Michel
- Le scope creep qui sprinte
- Les "comme Amazon" avec budget food truck
- Les analytics poussiéreux
- Les migrations "sans problème"
- Et 14 autres perles...
